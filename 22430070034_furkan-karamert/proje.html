İlerleme Çubuğu



Temel İlerleme Çubuğu

Bir kullanıcının bir süreçte ne kadar uzakta olduğunu göstermek için bir ilerleme çubuğu kullanılabilir. 


 
    < div  class="progress" > 
    < div class="progress-bar" style="width:70%" > < /div >  
    < /div >



İlerleme Çubuğu Yüksekliği

İlerleme çubuğunun yüksekliği varsayılan olarak 1rem şeklindedir. Boyutunu değiştirmek için CSS height özelliğini kullanılır. İlerleme kabı ve ilerleme çubuğu için aynı yüksekliği ayarlamamı gerekir. 


 
    < div  class="progress" > 
    < div class="progress-bar" style="width:40% ; height:20px" > < /div >  
    < /div >



İlerleme Çubuğu Etiketleri

Görünür yüzdeyi göstermek için ilerleme çubuğunun içine metin eklenir. 


 
    < div  class="progress" > 
    < div class="progress-bar" style="width:70%"  > 70% < /div >  
    < /div >

70%


Renkli İlerleme Çubukları

Varsayılan olarak ilerleme çubuğu mavidir. Rengini değiştirmek için bağlamsal arka plan sınıflarından herhangi birini kullanılır. 


 
    < div  class="progress" >
    < div class="progress-bar" style="width:10% " > < /div >  
    < /div >

    
    < div  class="progress" >
    < div class="progress-bar bg-success" style="width:20%" > < /div >  
    < /div > 

  
    < div  class="progress" > 
    < div class="progress-bar bg-info" style="width:30% " > < /div >  
    < /div > 
    

    < div  class="progress" > 
    < div class="progress-bar bg-warning" style="width:40% " > < /div > 
    < /div > 


    < div  class="progress" > 
    < div class="progress-bar bg-danger" style="width:50% " > < /div >  
    < /div >


    < div  class="progress" > >
    < div class="progress-bar bg-white" style="width:60% " > < /div >  
    < /div > 
    

    < div  class="progress" >
    < div class="progress-bar bg-secondary" style="width:70% " > < /div > 
    < /div >



Çizgili İlerleme Çubukları

"progress-bar-striped" kullanarak ilerleme çubuklarına şerit eklenir. 



    < div  class="progress" > 
    < div class="progress-bar progress-bar-striped" style="width:40% " > < /div >  
    < /div >



Çoklu İlerleme Çubuğu

İlerleme Çubukları çoklu kullanabilir. 



    < div  class="progress" >
    < div class="progress-bar bg-success" style="width:40% " > < /div >  
    Free Space < /div > 
    < div class="progress-bar bg-warning" style="width:10% " > < /div >  
    Warning < /div > 
    < div class="progress-bar bg-danger" style="width:20% " > < /div > 
    Danger < /div > 
    < /div >
Free Space
Warning
Danger


Hareketli İlerleme Çubuğu

"progress-bar-animated" kullanarak ilerleme çubuklarına şerit eklenir. 



    < div  class="progress" >
    < div class="progress-bar-striped progress-bar-animated" style="width:40% " > < /div > 
    < /div >




İplikçiler



İplikçiler

Bir döndürücü/yükleyici oluşturmak için ".spinner-border" sınıfı kullanılır. 


 

    < div  class="spinner-border" > < /div >


Renkli İplikçiler

Döndürücüye renk eklemek için herhangi bir metin rengi yardımcı programını kullanılır. 

 
    
        < div  class="spinner-border text-muted" > < /div >

        < div  class="spinner-border text-primary" > < /div >

        < div  class="spinner-border text-success" > < /div >

        < div  class="spinner-border text-info" > < /div >

        < div  class="spinner-border text-warning" > < /div >

        < div  class="spinner-border text-danger" > < /div >

        < div  class="spinner-border text-secondary" > < /div >



Büyüyen İplikçiler

".spinner-grow" Döndürücünün/yükleyicinin "döndürme" yerine büyümesini için sınıfı kullanılır. 


 
        < div  class="spinner-grow text-muted" > < /div >

        < div  class="spinner-grow text-primary" > < /div >

        < div  class="spinner-grow text-success" > < /div >

        < div  class="spinner-grow  text-info" > < /div >

        < div  class="spinner-grow text-warning" > < /div >

        < div  class="spinner-grow text-danger" > < /div >

        < div  class="spinner-grow text-secondary" > < /div >

        < div  class="spinner-grow text-dark" > < /div >

        < div  class="spinner-grow text-light" > < /div >



İplik Makinesi Boyutu

Daha küçük bir döndürücü oluşturmak için ".spinner-border-sm" veya ".spinner-grow-sm" öğesini kullanılır. 


 

    < div  class="spinner-border spinner-border-sm" > < /div >

    < div  class="spinner-grow spinner-grow-sm" > < /div >





Döner Düğmeler

Bir düğmeye metinli veya metinsiz olarak da döndürücüler eklenebilir. 




    < button class="btn btn-primary" > 
    
    < div  class="spinner-border spinner-border-sm" > < /div >

    < /button >
